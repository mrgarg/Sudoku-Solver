# Sudoku-Solver
solve the sudoku which is a partially filled 9x9 2D array with the help of recursion and backtracking

input is given in the form :-

1           # number of test cases 
3 0 6 5 0 8 4 0 0 
5 2 0 0 0 0 0 0 0
0 8 7 0 0 0 0 3 1 
0 0 3 0 1 0 0 8 0
9 0 0 8 6 3 0 0 5
0 5 0 0 9 0 6 0 0
1 3 0 0 0 0 2 5 0
0 0 0 0 0 0 0 7 4
0 0 5 2 0 6 3 0 0

#where zero denotes the unfilled values
